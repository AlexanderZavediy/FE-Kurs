Виды селекторов

  • А - В качестве селектора может выступать любой элемент HTML, для которого определяются правила форматирования, такие как: цвет, фон, размер и др.
  • #id - Идентификатор определяет уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты.
  • .class - Классы применяют, когда необходимо определить стиль для одного или нескольких элементов веб-страницы. Элементы при этом могут быть разными, но иметь частично или полностью похожие элементы оформления. В коде HTML имя класса определяется через атрибут class.
  • .class1.class2 - К любому элементу одновременно можно добавить несколько классов, перечисляя их в атрибуте class через пробел. В этом случае к элементу применяется стиль, описанный в правилах для каждого класса. Поскольку при добавлении нескольких классов они могут содержать одинаковые стилевые свойства, но сразными значениями, то берётся значение у класса, который описан в коде ниже.
  • A B - При создании веб-страницы часто приходится вкладывать одни элементы внутрь других. Чтобы стили для этих элементов использовались корректно, помогут вложенные селекторы. Можно одновременно установить стиль для отдельного элемента, а также для элемента, который находится внутри другого.

  • A > B - Дочерним называется элемент, который непосредственно располагается внутри родительского элемента.

    • Какой элемент выступает родителем, а какой его потомком легко выяснить с помощью дерева элементов — так называется структура отношений элементов документа между собой.
  • A + B - Соседними называются элементы веб-страницы, когда они следуют непосредственно друг за другом в коде документа .
