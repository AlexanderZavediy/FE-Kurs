  • git init - Создает пустой репозиторий Git или повторно инициализируйте существующий.
  • git status - Показывает статус рабочего дерева
  • git clone — клонирует репозиторий в новый каталог
  • git-add — добаляет содержимое файла в индекс
  • git-config — Получает и устанавливает репозиторий или глобальные параметры
  • git-commit - Записывает изменения в репозиторий
  • git-log — Показывает журналы коммитов
  • git-push — обноввляет удаленные ссылки вместе со связанными объектами
  • git-fetch — загружает объекты и ссылки из другого репозитория
